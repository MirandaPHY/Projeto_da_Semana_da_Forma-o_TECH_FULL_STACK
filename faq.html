Perguntas Frequentes

Aqui você encontra as respostas para as perguntas mais frequentes sobre o mundo da tecnologia.

1. O que é um computador?

Um computador é uma máquina capaz de variados tipos de tratamento automático de informações ou processamento de dados. Entende-se por computador um sistema físico que realiza algum tipo de computação. Assumiu-se que os computadores pessoais e laptops são ícones da era da informação.

2. O que é um smartphone?

Um smartphone é um celular que combina recursos de computadores pessoais, com funcionalidades avançadas que podem ser estendidas por meio de programas aplicativos executados pelo seu sistema operacional.

3. O que é um sistema operacional?

Um sistema operacional é um programa ou um conjunto de programas cuja função é gerenciar os recursos do sistema, fornecendo uma interface entre o computador e o usuário.

4. O que é um navegador de internet?

Um navegador de internet é um software que habilita os usuários a interagirem com documentos da web. Ele interpreta a linguagem HTML e apresenta o conteúdo e os aplicativos da web.

5. O que é um site responsivo?

Um site responsivo é um site que se adapta a qualquer dispositivo, seja ele um computador, tablet ou smartphone. Ele é capaz de se ajustar automaticamente ao tamanho da tela do dispositivo.
